Java 2D Geometry Library
A custom Java implementation for 2D geometric operations, developed for the SE 1108 course. This project focuses on Object-Oriented Programming (OOP) principles without using built-in geometry libraries.

Core Components
Point2D: Manages P(x,y) coordinates and calculates distances.

Line2D: Represents lines as y=mx+n and handles intersections and parallelism.

Path2D: Manages sequences of points, calculating total path length and concatenations.

Demo: A main class that validates all functionalities
